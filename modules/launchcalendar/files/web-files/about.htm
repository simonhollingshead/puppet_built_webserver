Launch Calendar · About

This subdomain hosts the result of an unofficial scraper that runs hourly against the SpaceFlightNow launch schedule. It attempts to parse the list of future launches, and creates an iCal file for subscribing to in your calendar software of choice.

If you're using Google Calendar, please be aware that they only check external calendars once or twice a day, so you may not get updates as quickly, and all unknown launches (which get set to today's date) won't be moved until many hours after midnight.

To find out more about the script, or see its source, please check my web server's GitHub Repository.

None of this would be possible without SpaceFlightNow (a website which I have no affiliation to). If you don't already use them for news related to space, you absolutely should. All data contained within the calendar entries is taken from their webpages, and is not owned by me at all.
